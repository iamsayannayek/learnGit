# Work flow for Git

## Step 1:

git clone "Repos link"

## Step 2:

Doing some changes

## Step 3:

Adding changes to git: git add .

## Step 4:

Commit changes: git commit -m "message for commit"

## Step 5:

Push to the Repo: git push -u origin main (For initial stage of pushing)
<br>
After that you only using git push
